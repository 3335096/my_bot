# Telegram OpenRouter Agent Bot

Личный Telegram-бот-агент на Python (`aiogram`) + OpenRouter.

Поддержка:
- текстового общения;
- голосовых сообщений (транскрибация + ответ);
- изображений (vision-анализ);
- умного роутинга задач по типу запроса;
- истории диалогов: последние 10, сохраненные, принудительное удаление.

## Быстрый старт

1. Создайте `.env` из шаблона:

```bash
cp .env.example .env
```

2. Заполните обязательные переменные:
- `TELEGRAM_BOT_TOKEN`
- `OPENROUTER_API_KEY`
- `DATABASE_URL`

3. Установите зависимости и запустите:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -e .
python -m bot.main
```

## Команды

- `/start` — старт и подсказка
- `/help` — список команд
- `/new` (или `/reset`) — новый диалог
- `/history` — последние 10 диалогов
- `/saved` — сохраненные диалоги

## UX-правила (зафиксировано)

- Routing-бейдж показывается **только в первом ответе новой сессии**.
- Последние диалоги: строго **10**.
- В списке последних: `Открыть`, `Сохранить`, `Удалить`.
- В списке сохраненных: `Открыть`, `Удалить`.
- Лимиты хранения:
  - `RECENT_SESSIONS_LIMIT=10`
  - `SAVED_SESSIONS_LIMIT` (по умолчанию 50)
- При превышении лимита удаляется самый старый диалог из соответствующего списка.
- Кнопка удаления удаляет диалог принудительно сразу.
